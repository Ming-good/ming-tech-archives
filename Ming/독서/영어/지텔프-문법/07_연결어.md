## 필수문법
#### 접속부사
- , 앞에 위치 / 없어도 문법적 오류 없음
```
He is handsome. However, I don't like him.
```
#### 접속사
- 절과 절을 연결 / 없으면 틀린 문장
```
Althought he is handsome, I don't like him.
```
#### 접속부사/접속사 구분
```
Because I studied hard, I passed the test.
I studied hard. Nevertheless, I failed to pass the test.
I passed the test althought I didn't study hard.
I studied hard. Therefore, I passed the test
```
#### 전치사
- 명사(구) 앞에 위치
```
Despite the rain, we went on a picnic.
```
#### 역접과 순접
- 역접: 앞의 글과 반대되거나 일치하지 않는 서술이 뒤에 나옴
```
Althought she is rich, she is not happy.
```
- 순접: 앞문장과 뒷문장이 논리적으로 모순되지 않음
```
Because she studied hard, she passed a test.
Because she didn't study hard, she failed a test.
```
## Point
#### 접속부사
- 없어도 문법적 오류 없음
```
She doesn't have money. [Nevertheless / Although], she is happy
```
- 해석 필요
```
He is handsome. [However / Therefore], I like him.
```
#### 접속사
- 절과 절을 연결(꼭필요)
```
[Nevertheless / Althought] she doesn't have money, she is happy
```
- 해석 필요
```
[Althought / Because] he is handsome, I like him.
```
#### 전치사(명사 앞에 위치)
```
Despite the rain, we went on a picnic.
= In spite of the rain, we went on a picnic.
```
