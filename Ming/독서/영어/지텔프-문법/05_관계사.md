## 필수 문법
#### 관계사가 속한 문장: 앞에 있는 명사 수식 (해석 ~ㄴ)
```
I have a book which is interesting
```
#### 선행사: 관계사 앞에 있는 명사
```
사물: I have a book which is interesting
사람: I know a girl who is pretty.
장소: This is a building where I study English.
시간: I recomembered the day when i had met him.
```
#### 문장 불완전 완전
-  관계사 뒷문장 불완전: 주어 + 타동사 + 목적어 없음, 주어없음 + 동사
-  관계사 뒷문장 완전: 주어 + 타동사 + 목적어, 주어 + 자동사
```
I have a book which is interesting. [불]
I know a girl who is pretty. [불]
I have two cups which are expensive. [불]
This is a building where I study English. [완]
I met a guy whom she liked. [불]
```
> 선행사가 불완전한 문장의 원래 주어나 목적어였을 것!
- 자동사
```
This is a house where I live

주요 자동사: live, wait, come, go, happen, occur, work, appear 등
```
## Point
#### 사람 선행사 + 뒤에 주어없음 => who, that
```
I like a girl [who / which / where] hate me.
```
#### 사람 선행사 + 뒤에 목적어 없음 => whom, who ,that
```
I know a girl [who / which / whom] he likes.
```
#### 사물 선행사 + 뒤에 주어나 목적어가 없음 => which, that
```
주격관계대명사: I have a book [which / who / that] is  interesting.
목적격관계대명사: I have a book [which / who] she likes.
I have a book [which likes her / that she likes]
```
#### 선행사 존재(뒤에 나오는 주어와 소유 관계) + 뒷문장 완전 => whose
```
I know a girl [who / whose] computer is expensive
```
#### , 뒤 => that 소거
```
I have the bookm [which, that] is interesting.
```
### 뒷문장 완전 => where(장소), when(시간)
```
장소: This is the cafe [which / when / where] I met him.
시간: I remember the day [which / when / where] i met him.
```
## 팁!
1. , 뒤 that 소거
2. 빈칸 앞 단어 사물인지 사람인지 파악
