## 필수 문법
#### To 부정사란
- to + 동사원형
```
I studied English to pass the test.
```
- to 부정사의 다양한 자리
```
1) I planned to study. [목적어 자리: 동사 뒷자리]
   S + V + O
2) I studied English to pass the test. [부사자리: ~하기 위해서]
   S + V + O + 부사
3) I will buy a book to read [형용사 자리: ~할, ~하는]
   S + V + O + 형용사
```
> 형용사: 꾸며주는 말, 부사: 없어도되는 말
#### 동명사
- v-ing(것)
```
I enjoyed studying English,
```
- 동명사의 다양한 자리
```
1) Studying English is difficult. [주어자리]
2) I enjoyed studying English. [목적어 자리: 동사 뒷자리]
3) It prevented her from stdying. [전치사 뒷자리]
```

## Point
#### 전치사 뒤 빈칸 ing(동명사)
```
She is responsible for [to handle / handling] the issues.
```
#### 주어 자리  빈칸 ing(동명사)
```
[Taking / To take] care of her is hard.
```
#### 동사 뒤 목적어 자리 빈칸 ing(동명사)
```
I enjoy [studying / to study] English.
```
#### 동사 뒤 목적어 자리 빈칸 to부정사
```
I planned [studying / to study] English.
```
#### 동사 뒤 to부정사 ing 둘 다 가능(해석필요)
```
The doctor recommended that she stop [smoking / to smoke]
```
#### 부사 자리에 빈칸(~하기 위하여로 해석) To 부정사
```
I study English [passing / to pass] the test.
```
#### 형용사 자리 빈칸(~할, ~하는으로 해석) To 부정사
```
I have a few friends [trusting / to trust].
```
### 주어 자리에 가주어 it이 있고, 진주어 자리에 빈칸 To부정사
```
It is important [studying / to stydy] English.
```
#### 암기해야 할 to 부정사 표현
- to 부정사를 목적보어로 가지는 경우 (S + V + O=OC)
```
She allowed me [using / to use] her computer.
```
- be/been/being + pp + to부정사
```
I was allowed [use / to use] her computer.
```
- 동명사의 관용적 표현
```
She couldn't help [studying / to study] English.
```
## 오답노트
- `allow, advise, encourage, recommend, require`
	- 목적어와 목적보어에 오는 준동사의 형태가 다르다
```
She allowed [to meet / meeting] him. => 목적어자리는 ing
She allowed them [to stay / staying] here. => 목적어 + to 부정사
```
- be(been,being) allowed, advised, encouraged, recommended required + to 부정사
```
She was allowed [to stay / staying] here.
```
- bee pp 뒤에 빈칸이 나온다고 하여 무조건 to 부정사가 정답은 아니다
```
(be caught + ing)
He was caught smoking
```

## 팁!
- ing 또는 to부정사만 정답
- ing 3개  + to 부정사 3개
