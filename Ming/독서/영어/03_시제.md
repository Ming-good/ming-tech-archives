## 문제특징
#### 출제 문제수: 6
- 과거진행1: was/ were ~ing
- 현재진행1: am/is/are ~ing 
- 미래진행1: will be ~ing
- 과거완료진행1: had been ~ing
- 현재완료진행1: have(has) been ~ing
- 미래완료진행1: will have been ~ing
#### 선택지 특징
1. 동사 위주
2. 무조건 ~ing로 끝나는 선택지

## 필수 문법

- 과거진행형: ~하고 있는 중이었다
- 현재진행형: ~하고 있는 중이다
- 미래진행형: ~하고 있는 중일 것이다.
- 현재완료 진행형: ~해오고 있는 중이다.
	- (과거of과거) from (과거) 
- 과거완료 진행형: ~해오고 오고있는 중이었다.
	- (과거) from (현재)
- 미래완료 진행형: ~해오고 있는 중일 것이다.
	- (현재) from (미래)

## Point
![[Pasted image 20260408221533.png]]
#### Point 1
> 주절과거 + while (as) 과거 진행형
```
I ate something while (as) she [is writing / was writing]
```
#### Point 2
> when 과거 + 주절과거 진행형 (**for / since**가 있을경우 과거완료진행형이다!)
```
I [am reading / was reading] a book when she wrote a letter. 
```
#### Point 3
> 현재힌트 => 현재 진행
- 선택지에 now가 존재하는 경우 (**for/since**가 있을경우 현재완료진행형이다.)
#### Point 4
1. 미래힌트
```
They [were studying / will be studying] English tomorrow.
```
1. 시간조건 부사절+현재시제 -> 미래진행형
```
If she comes, Jack [has studied /  will be studying] English.
```
#### Point 5
1. 과거 힌트 + for,since
```
I [had been reading / was reading] a book for 2 hours when she wrote a letter.
```
1. before 과거 시제 => 과거완료 진행형
```
I [had been sudying / was studying] before he called me.
```
#### Point 6
> 현재진행형 힌트 + for, since => 현재완료 진행형
```
She [is reading / has been reading] a book now for 2 hours.
```
#### Point 7
> 미래진행형 힌트 + for, since => 미래완료 진행형
```
I [had been reading / will have been reading] a book for 2 hours by the time he comes here.
```
## 오답노트
1. for가 있다고 무조건 완료진행 시제가 정답은 아니다
```
정답: She has been studying for 3 hours.
오답: I will be buying flowers for you.
```
> for + 시간일 경우만 완료진행 시제의 힌트로 기능
2.  since 뒤에 과거시점이 나오는 경우 현재완료진행형 가능
```
현재진행완료: I have been studying English since yesterday afternoon.
과거진행완료: I had been studying English since yesterday afternoon.
=> 힌트: But I failed the test
```
>  만약에 현재진행완료와 과거진행완료가 나온다면 해석을 통해 풀어야한다.
