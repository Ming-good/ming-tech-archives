## 학습 전 알아야할 필수 문법
#### 동사원형
- Should의 뜻: ~해야 한다
- Should는 조동사: 무조건 뒤에 동사 원형
```
원문: She suggested that he should [goes/go] there
생략: She suggested that he [goes/go] there
```
- be 동사의 원형:be(is는 못나오지만 be는 가눙!)
#### 동사의 능동과 수동
- 능동: '~하다'
- 수동: '~되다, 당하다' 등 `[be + pp]`
- 동사의 능동과 수동의 결정: 주어
>보통 능동은 동사 뒤에 목적어(명사) 있음
  보통 수동은 동사 뒤 목적어(명사) 없음
## 주장,요구,명령,제안,결정,충고와 관련된 단어
#### 1. "이렇게 하는 게 어때?" (제안·권고 그룹)

| **단어**                         | **의미**        |
| ------------------------------ | ------------- |
| **Suggest / Suggestion**       | 제안하다 / 제안     |
| **Propose / Proposal**         | 제안하다 / 제안     |
| **Recommend / Recommendation** | 권고(추천)하다 / 권고 |
| **Advise**                     | 충고(조언)하다      |
| **Desirable**                  | 바람직한          |
| **Best**                       | (하는 것이) 최선인   |
#### 2. "이거 꼭 해야 해!" (요구·요청 그룹)

|**단어**|**의미**|
|---|---|
|**Ask / Request**|요청하다|
|**Require / Requirement**|요구하다 / 요건(필요조건)|
|**Demand**|(강력히) 요구하다|
|**Desire**|간절히 바라다(열망하다)|
|**Urge / Urgent**|촉구하다 / 긴급한|
|**Lobby**|(압력을 가해) 로비하다|
#### 3. "무조건 해, 내 말 들어!" (주장·명령 그룹)
|**단어**|**의미**|
|---|---|
|**Insist**|주장하다|
|**Order / Command**|명령하다|
|**Determine / Decision**|결정하다 / 결정|
|**Stipulate / Prescribe**|명시(규정)하다 / 규정하다|
|**Decree / Rule**|(법령으로) 정하다 / 판결(지배)하다|
|**Stress**|강조하다|
|**Claim**|(권리를) 주장하다|
#### 4. "안 하면 큰일 나!" (의무·중요성 형용사)
|**단어**|**의미**|
|---|---|
|**Essential / Necessary**|필수적인 / 필요한|
|**Mandatory / Compulsory**|의무적인 / 강제적인|
|**Important / Vital**|중요한 / 치명적으로 중요한|
|**Crucial / Imperative**|중대한 / 긴요한(필수인)|
|**Natural**|당연한|
> 주로 `It ie [혛용사] that...`구조로 쓰이며 당위성을 강조한다.
## POINT
#### Point 1
- '주요명제결충'과 관련된 동사, 명사, 형용사: that절에 동사원형
```
1) He insisted that he [study / studies] English.
2) It is essential that you [obey / obeyd] your parents.
```
#### Point 2
- 선지에 should와 동원이 둘 다 존재하는 경우: 동원이 답
```
He recommended that she [should go / go].
```
#### Point 3
- 동사의 태를 구분해야 하는 경우: 주어 찾아 증/수동 구분.
```
1) A teacher suggested that the programs [install / be installed].
2) It is best that the measure [take / be taken].
3) They insisted that wildlife [protect / be protected].
```
