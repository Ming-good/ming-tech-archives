## 출제 문제
- 가정법 과거3 + 가정법 과거완료3
## 가정법 파악 방법
1. 동사 위주로 구성
2. 보통 would / could / might로 시작한다

## 가정법 과거
#### Point 1 
> `if 주어 + 과거동사(v-ed),주어 + w/c/m + 동사원형`
```
 if i studied hard, I [would have passed / would pass] the test
 = I [would have passed / would pass] the test  if i studied hard.
```
- 가정법 과거 : 해석  현재(만약 ~한다면, ~할텐데)
#### Point 2
>  `if주어 + had pp, 주어 + w/c/m + have pp`
```
if I had studied hard, I [would have passed / would pass] the test
= I [would have passed / would pass] the test if I had studied hard.
```
- 가정법 과거완료: 해석 과거(만약 ~했다면, ~했을텐데)

## 가정법 오답
#### Point 1  
> `Had + 주어 ~` : 가정법 과거완료
- 도치 : if의 의미가 있다는 것을 표현하기 위해 문장 구성을 뒤 바꾸는 것
```
(if) (S <=> had) pp, S w/c/m have pp  

Had he studied hard Chris [would pass / would have passed] the test.
= Chris [would pass / would have passed] the test had he studied hard.

```
#### Point 2 
> `if 주어 + had(과거V) + 명사` : 가정법 과거
- had 뒤에 pp가 나왔다면 과거분사로 판단 = 즉, `w/c/m have pp`
- had 뒤에 명사가 나왔다면 과거동사로 판단  = 즉, `w/c/m 동사원형`\
```
if I had more time, I [could do / could have done] it.
```
#### Point 3 
>`if 주어 + could / were / did ~` : 가정법 과거
```
I [would support / would have supported] her if Rachel could make it
```
#### Point 4 
> `if 주어 + 현재 시제`: 가정법 XX
```
if I wear the funny dress, she will be laughing at me
```
#### Point 5 
> `동사 뒤의 if가 '~인지 아닌지'로 해석될 떄'`: 가정법 X 
- see, ask, wonder, check, decide 등 일 경우, 주의 깊게 해석 
```
I wonder if she will meet him
```
#### Point 6
> `if 주어 + had pp, 주어 + w/c/m + 동사원형 (right) now` : 혼합 가정법
```
If Rachel had not missed the train, she would be in Seoul right now.
```
