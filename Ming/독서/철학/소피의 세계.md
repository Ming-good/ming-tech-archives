7. **소크라테스**
	- 제자 : 플라톤
	- 유명한 말 : 
		- 나는 아테네를 게으른 암말에 비유할 수 있고, 나는 신이 그 말에 붙여준 등에와 같다. 나는 이 말을 깨우고 활동적이게 하기 위해 옆구리를 계속 찌른다. 나를 없애면 아테네는 더 깊은 나태에 빠질 것이다.
		- 내가 알고 있는 단 한 가지는, 내가 아무것도 모르고 있다는 사실이다.
	- 문답법 자주 사용
	- 철학자(Philosophs) : 지혜를 사랑하는 사람. 이해하지 못한 것이 아주 많다는 사실을 깨닫는 사람
	- 소크라테스는 옳고 그름을 구별하는 능력은 소피스트들과 달리 사회가 아니라 인간 이성에 있다고 믿음.
		- 무엇이 옳고 그른지 **보편타당한 개념** 정의를 내리는 것은 중요
		- 옳지 않은 행동을 하는 사람은 그들 역시 알고 있을 것.
		1. 자신의 확신에 반대되는 행동을 하는 사람은 행복해질 수 없다.
		2. 어떻게 해야 행복해질 수 있는지 아는 사람은 그런 사람이 되려고 노력한다.
		3. 무엇이 옳은지 아는 사람은 옳은 일을하게 된다.

9. **아리스토텔레스**
	- **아리스토텔레스가 유럽 문화에서 갖는 의미** : 오늘날까지도 사용되는 여러 학술어를 만들어냈으며 여러 학문을 기초하고 계통을 세운 위대한 체계 조직자
	- **플라톤과 반대되는 생각** : 형상에 대해 사람의 영혼 속에 있는 것은 단지 자연적 대상의 반영(이데아론의 반대). 즉, 감각세계에서 지각하기 전까지 우리는 빈 상태이기 때문에 선천적인 이데아도 없는 것이다.
	- **형상과 질료** :
		- **형상** : 사물의 특성
		- **질료** : 사물을 이루는 재료
	- **4가지의 원인** :
		- **목적원인** : 깨진 창문을 보면 왜 창문이 깨졌는지, 누가 돌을 던졌는지 하는 것 뿐만 아니라 왜 던졌는지
		- **질료원인** : 비의 재료라는 원인 혹은 질료원인은 대기가 차가워졌을 때 바로 거기에 수증기(구름)라는 질료가 있기 때문.
		- **작용원인** : 수증기를 냉각시키는 일
		- **형상 원인** : 땅에 떨어지는 것이 물의 본성 혹은 물의 형상(특징)
	- **존재론적 생각** : 자연의 모든 것에는 합목적성이 깃들어 있다고 믿음
	- **논리학** : 
		- 아리스토텔레스는 자연을 철저하게 정돈하려고함
		- 아리스토텔레스 놀이 : 특징을 물어보고 상위 그룹부터 하위 그룹으로 특정 대상의 범위를 축약하며 대상이 무엇인지 찾는 게임.
		- 대상이 하위 그룹에 포함되어 있다면 연역법적 추론을 통해 대상이 상위 그룹의 특징을 가지고 있음을 추론할 수 있음.
	- **자연의 사다리** :
		- 자연현상들을 여러 다른 그룹으로 구분했을 때 기준 삼었던 것은 사물들의 특성, 정확히는 각 사물이 할 수 있는 일과 작용
	- **윤리학** : 
		- 균형과 절제를 통해서만 행복하고 조화로운 인간이 될 수 있다.
	- **정치학** : 아리스토텔레스가 생각하는 여러가지 국가의 형태
		- 군주제 : 군주국가가 훌륭하게 실현되려면 한 사람의 지배자가 자기 이익을 위해 국가를 주무르는 독재정치로 전락하면 안 됨.
		- 귀족제 : ****
			- 크고 작은 무리의 통치자들이 존재
			- 노늘날 군사혁명 위원회 같은 것에서 소수 인물이 통치권을 장악하는 것을 경계해야 됨.
