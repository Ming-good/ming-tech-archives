📂 그리디 알고리즘 유형 정리 (PDF 스타일 요약)

────────────────────────────

✅ 1. 구간 덮기 / 최소 커버 문제  
────────────────────────────
- 대표 문제: 공주님의 정원(2457), 선 긋기(2170)
- 핵심 아이디어:  
    • 시작일 기준 오름차순 정렬  
    • 현재 날짜까지 피는 꽃들 중 지는 날이 가장 늦은 것을 선택  
    • 커버 불가능한 공백이 생기면 실패
- 적용 예시:  
    • 행사 기간 커버, 유지보수 인력 최소화, 인터벌 합치기 등

✅ 2. 최대한 많이 선택 문제  
────────────────────────────
- 대표 문제: 회의실 배정(1931), 강의실 배정(11000)
- 핵심 아이디어:  
    • 종료 시간을 기준으로 오름차순 정렬  
    • 가능한 한 겹치지 않게 많이 배치
- 적용 예시:  
    • 일정 최적화, 자원 분배, 배차 스케줄링

✅ 3. 최대/최소값 만들기 문제  
────────────────────────────
- 대표 문제: 잃어버린 괄호(1541), 주식 문제
- 핵심 아이디어:  
    • 연산 순서를 바꿔서 값 최적화 (예: '-' 기준으로 그룹 묶기)  
    • 이득이 생기는 시점을 중심으로 선택
- 적용 예시:  
    • 식 최적화, 금액 절감, 리스크 최소화

✅ 4. 우선순위 큐를 활용한 선택 최적화  
────────────────────────────
- 대표 문제: 컵라면(1781), 보석 도둑(1202), 강의실 배정(11000)
- 핵심 아이디어:  
    • 조건 만족하는 후보들 중 최대 가치 항목 선택  
    • 힙(우선순위 큐)을 활용해 실시간 정렬 처리
- 적용 예시:  
    • 제한된 자원으로 최대 수익, 시간 제한 최적화 문제

✅ 5. 미래 예측 기반 교체 문제  
────────────────────────────
- 대표 문제: 멀티탭 스케줄링(1700), 캐시 교체 알고리즘 등
- 핵심 아이디어:  
    • 앞으로 가장 늦게 다시 사용될 항목을 제거  
    • 미래를 예측해서 현재의 최적 결정을 내림
- 적용 예시:  
    • 캐시 메모리 관리, 자원 교체 최적화
    

📘 연습 문제 추천 리스트  
────────────────────────────
1. 회의실 배정 (1931) - 기초 그리디의 정석
2. 공주님의 정원 (2457) - 구간 덮기 마스터
3. 선 긋기 (2170) - 겹치는 구간 처리 연습
4. 컵라면 (1781) - 마감일과 이득의 조합
5. 보석 도둑 (1202) - 무게 제한, 최대 이득
6. 멀티탭 스케줄링 (1700) - 교체 전략
    

✅ 실전 팁:
- 그리디는 항상 "지금 최선의 선택이 전체적으로 최선인가?"를 검증해야 함
- 정렬 기준을 잘 잡는 것이 절반 이상
- 실패 조건(덮이지 않는 구간 등)을 명확히 설정할 것
