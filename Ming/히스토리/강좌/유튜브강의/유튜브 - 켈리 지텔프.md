- [[01_가정법]]
	- 가정법 과거 : (wcm동사원형)
	- 가정법 과거완료  : (wcm have pp)
	- 오답
		- 도치현상
		- `had 명사` : 가정법 과거
		- `could/were/did` : 가정법 과거
		- if + 주어 + 현재시제 : 가정법 아님
		- if가 '~인지 아닌지'로 해석 : 가정법 아님
		- 혼합가정법 : 해석을 통해 문제 풀어야됨
- [[02_SHOULD 생략]]
	- '주요명제결충'과 관련된 동사, 명사, 형용사: that 절에 동사원형
	- 오답
		- 선지에 should + 동원과 동원이 둘다 존재하는 경우: 동원
		- 동사의 태를 구분해야하는 경우
		- 주어 찾아 능/수동 구분
