- [[01_가정법]]
	- 가정법 과거 : (wcm동사원형)
	- 가정법 과거완료  : (wcm have pp)
	- 오답
		- 도치현상: 가정법 과거완료
		- `had 명사` : 가정법 과거
		- `could/were/did` : 가정법 과거
		- if + 주어 + 현재시제 : 가정법 아님
		- if가 '~인지 아닌지'로 해석 : 가정법 아님
		- 혼합가정법 : 해석을 통해 문제 풀어야됨
- [[02_SHOULD 생략]]
	- '주요명제결충'과 관련된 동사, 명사, 형용사: that 절에 동사원형
	- 오답
		- 선지에 should + 동원과 동원이 둘다 존재하는 경우: 동원
		- 동사의 태를 구분해야하는 경우
		- 주어 찾아 능/수동 구분

	- 현재진행형

	- 미래진행형

		- 과거 힌트 + for, since

- [[05_관계사]]
	- 사람 선행사 + 뒤에 주어 없음: who, that
	- 사람 선행사 + 뒤에 목적어 없음: whom, who, that
	- 사물 선행사 + 뒤에 주어나 목적어 없음: which, that
	- 선행사존재(뒤에 나오는 주어와 소유 관계) + 뒷문장 완전: whose
	- ,뒤 that 소거
	- 뒷문장 완전 => where(장소), when(시간)
	- what은 선행사 없어야됨.
- [[06_조동사]]
	- can => 능력,가능성,허가
	- should => 충고, 권유, 도덕성
	- must => 의무
	- will => 미래에 있을 확실한 일, 의지
	- may => 약한 추측
	- 팁
		- Shall 답 X
		- 동물 주어는 can
		- (in order)to 부정사 should, must
