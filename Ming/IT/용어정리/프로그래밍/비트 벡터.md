## 1. 비트 벡터란?
>하나의 정수 타입 (`long`, `long[]`)의 **비트 자리수**(0과 1)를 이용해서  
>각 enum 값이 Set에 포함되었는지를 **기록**하는 방식
## 2. 구조와 동작 방식
```
enum Day {
    MONDAY,    // ordinal = 0
    TUESDAY,   // ordinal = 1
    ...
    SUNDAY     // ordinal = 6
}

EnumSet<Day> set = EnumSet.of(Day.MONDAY, Day.WEDNESDAY, Day.SATURDAY);

```

|enum|ordinal|비트 위치|값|
|---|---|---|---|
|MONDAY|0|0000001|1|
|TUESDAY|1|0000010|0|
|WEDNESDAY|2|0000100|1|
|THURSDAY|3|0001000|0|
|FRIDAY|4|0010000|0|
|SATURDAY|5|0100000|1|
|SUNDAY|6|1000000|0|
## 3. 포함 여부 확인 방식
```
boolean contains = (bitVector & (1 << ordinal)) != 0;
```
## 4. 장점
|항목|설명|
|---|---|
|빠른 속도|비트 연산으로 `O(1)` 처리 (contains, add, remove 등)|
|메모리 절약|64개 이하 enum이면 `long` 하나로 표현 가능 (8바이트)|
|순서 유지|enum 정의 순서대로 비트 위치가 정해지므로 순서 일관성 유지|
