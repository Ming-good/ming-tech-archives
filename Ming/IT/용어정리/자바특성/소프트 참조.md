```
ArrayList<String> orgObj = new ArrayList<>();  
SoftReference<ArrayList> copyObj = new SoftReference<>(orgObj);
```
대상 객체를 참조하는 경우가 SoftReference 객체만 존재하는 경우 GC의 대상이 된다.
단, JVM 메모리가 부족한 경우에만 힙 영역에서 제거되고 메모리가 부족하지 않다면 굳이 제거하지 않는다.

```
ArrayList<String> orgObj = new ArrayList<>();  
orgObj.add("테스트2");  
SoftReference<ArrayList> copyObj = new SoftReference<>(orgObj);  
  
orgObj = null;  
System.gc();  
  
System.out.println("원본 : " + orgObj);  
System.out.println("약한참조 : " + copyObj.get());  
System.out.println("ArrayList 객체 자원이 해소 되었다.");

----

[결과]
원본 : null
약한참조 : [테스트2]
ArrayList 객체 자원이 해소 되었다.
```
