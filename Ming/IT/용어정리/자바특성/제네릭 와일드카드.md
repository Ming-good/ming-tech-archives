이 물음표가 와일드 카드이며, 물음표는 어떤 타입이든 될 수 있다는 뜻을 가지고 있다.

|     와일드 카드      |                   네이밍                    |                       설명                       |
| :-------------: | :--------------------------------------: | :--------------------------------------------: |
|      \<?\>      |   Unbounded wildcards  <br>비한정적 와일드 카드   |               제한 없음 (모든 타입이 가능)                |
| \<? extends U\> | Upper Bounded Wildcards  <br>상한 경계 와일드카드 | 상위 클래스 제한 (U와 그 자손들만 가능)<br>상한이 U라 상한 경계라고 한다. |
|  \<? super U>   | Lower Bounded Wildcards  <br>하한 경계 와일드카드 | 하위 클래스 제한 (U와 그 조상들만 가능)<br>하한이 U라 하한 경계라고 한다. |


## 상한 경계 와일드카드([[공변성]])
![[Pasted image 20250216122123.png]]
```
class MyArrayList<T> {
    Object[] element = new Object[5];
    int index = 0;
    public MyArrayList(Collection<? extends T> in) {
        for(T elem : in) {
            element[index++] = elem;
        }
    }
}

public static void main(String[] args) {
    MyArrayList<Number> list;
    Collection<Integer> col = Arrays.asList(1, 2, 3, 4, 5);
    list = new MyArrayList<>(col);
    
    System.out.println(list); // [1, 2, 3, 4, 5]
}
```


## 하한 경계 와일드카드([[반공변성]])
![[Pasted image 20250216123541.png]]```
```
class MyArrayList<T> {
    Object[] element = new Object[5];
    int index = 0;
    public MyArrayList(Collection<? extends T> in) {
        for (T elem : in) {
            element[index++] = elem;
        }
    }
    public void clone(Collection<? super T> out) {
        for (Object elem : element) {
            out.add((T) elem);
        }
    }

    @Override
    public String toString() {
        return Arrays.toString(element); 
    }
}

public static void main(String[] args) {
    MyArrayList<Number> list = new MyArrayList<>(Arrays.asList(1, 2, 3, 4, 5));

    List<Object> temp = new LinkedList<>();
    temp = list.clone(temp);

    System.out.println(temp); // [1, 2, 3, 4, 5]
}
```

## 비한정적 와일드카드
제네릭에 extend, super 범위를 제한하지 않고 심플하게 \<?\>비한정적 와일드카드로 구성하게되면 **어떠한 타입이도 올 수 있지만, 동시에 어떠한 타입이든 올 수 있는 문제**로 매개변수를 꺼내거나 저장하는 로직에서 **논리적 에러가 발생한다.**
