```
MyClass obj = new MyClass();
```

obj 변수가 참조를 가지고 있는 한, MyClass 객체는 GC의 대상이 되지 않는다.
```
ArrayList<String> orgObj = new ArrayList<>();  
orgObj.add( "test1");  
  
ArrayList copyObj = orgObj;  
  
orgObj = null;  
  
System.gc();  
  
System.out.println("원본 : " + orgObj);  
System.out.println("강한참조 : " + copyObj);  
System.out.println("ArrayList 객체 자원이 해소 되지 않는다.");

-----
[결과]
원본 : null
강한참조 : [test1]
ArrayList 객체 자원이 해소 되지 않는다.

```
