```
ArrayList<String> orgObj = new ArrayList<>();  
WeakReference<ArrayList> copyObj = new WeakReference<>(orgObj);
```

대상 객체를 참조하는 경우가 WeakReference 객체만 존재하는 경우 GC의 대상이 된다.
다음 GC 실행 시 무조건 힙 메모리에서 제거된다.

```
ArrayList<String> orgObj = new ArrayList<>();  
orgObj.add("테스트2");
WeakReference<ArrayList> copyObj = new WeakReference<>(orgObj);  
  
orgObj = null;  
System.gc();  
  
System.out.println("원본 : " + orgObj);  
System.out.println("약한참조 : " + copyObj.get());  
System.out.println("ArrayList 객체 자원이 해소 되었다.");

----

[결과]
원본 : null
약한참조 : null
ArrayList 객체 자원이 해소 되었다.
```
