## 기능
- 동의어를 고려한 검색
- 오타를 고려한 검색
- 검색어 자동 완성 기능
- 검색어와 가장 관련성이 높은 순서대로 데이터를 조회하는 기능

## 명령어
```json
// 인덱스 생성
PUT /news_feed

// 인덱스 조회
GET /news_feed/_search

// 인덱스는 유지한 채로 필드만 변경
PUT /news_feed/_mapping
{
  "properties": {
    "title": { "type": "text", "analyzer": "nori" },
    "content": { "type": "text", "analyzer": "nori" },
    "tags": { "type": "keyword" }
  }
}

// 필드 삽입
POST /news_feed/_doc
{
  "title": "SpringDataJPA(스프링데이터JPA)를 뜯어보자 ",
  "content": "Spring Data Jpa는 SQL 쿼리를 직접 작성하는 대신 객체와 객체 간의 관계에 집중할 수 있게 돕는 기술, 덕분에 Java의 ORM 기술인 JPA (Java Persistence API)를 더 쉽고 편리하게 사용",
  "tags": "IT"
  
}

// ID를 지정하며 필드 삽입
POST /news_feed/_create/1
{
  "title":"공통 인터페이스",
  "content": "스프링 데이터 JPA를 사용하려면 JpaRepository를 상속받은 공통 인터페이스만 생성하면 됨",
  "tags": "IT"
}

// UPDATE + INSERT
PUT /news_feed/_doc/2
{
  "title": "공통 인터페이스3",
  "content": "그러면 스프링 데이터 JPA가 구현 클래스 대신 만들어줌 (공통 CRUD를 자동으로 생성) ",
  "tags": "IT"
}

// ID를 이용해 조회
GET /news_feed/_doc/2
```
