## 애널라이저란?
- 애널라이저란 문자열을 토큰트로 변환시켜주는 장치를 보고 애널라이저라고 부른다.
![[Pasted image 20260305170527.png]]
#### 캐릭터 필터(character filter)
- **역할**: 문자열을 토큰으로 자르기 전에 문자열을 다듬는 역할을 한다.
	- html_strip필터 적용(HTML 태그 제거)
	- mapping필터
	- pattern replace필터
#### 토큰나이저(tokenizer)
- 역할: 문자열을 토큰으로 자르는 역할
	- standard 토크나이저(공백 또는 `,`,`.`,`!`,`?`와 같은 문장 부호를 기준으로 자름)
#### 토큰 필터(token filter)
- 역할: 잘린 토큰을 최종적으로 다듬는 역할을 한다.
- 다양한 종류의 필터가 존재하며, 여러 개의 필터를 적용할 수 있다
	- lower 필터 적용 (소문자로 변환)
	- stop 필터 적용 (a, the, is와 같은 특별한 의미를 가지지 않는 단어를 제거)
	- stemmer 필터 적용 (단어의 원래 형태로 변환)
		- brown -> brown
		- foxes -> fox
		- jumped -> jump
---
## DEFAULT 애널라이저
- 구성:
	- 캐릭터 필터: X
	- 토크나이저: standard tokenizer
	- 토큰 필터: lowercase
- 표현방식:
```json
## 방법1
"analyzer": "standard"

## 방법2
"char_filter": [],
"tokenizer": "standard",
"filter": ["lowercase"]
```
