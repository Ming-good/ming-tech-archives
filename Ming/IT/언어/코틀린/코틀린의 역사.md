Kotlin은 JetBrains에서 2011년에 공개한 언어로 자바와 100퍼센트 호환되며 자바보다 더 간결하고 많은 기능을 추가한 언어이다.
## Kotlin이 개발된 이유
JebBrains가 새언어를 만든 이유는 본인들이 불편해서 만들게 됨.
자바의 보일러플레이트 코드 때문에 개발 속도가 안 났고 따라서, JebBrains사는 자바보다 간결하고 편리한 Kotlin을 개발하게 됨.
- 자바의 상위 호환 버전
	- 자바 기능과 더불어 코틀린의 새로운 기능까지 모두 사용 가능함
- 보일러플레이트 코드를 줄여서 유연하고 수정도 빠르다

## Kotlin을 사용하는 이유
- **Null Safety (안전성):**  `NullPointerException`을 언어 차원에서 방지가 가능하다
- **간결함 (Conciseness):** 자바에서 수십 줄 써야 했던 `Getter/Setter`, `toString`, `equals` 등을 `data class` 한 줄로 끝낼 수 있다
- **100% 자바 호환성 (Interoperability):** 자바 프로젝트와 코틀린을 섞어서 사용해도 전혀 문제가 없으며 기존 자바 라이브러리를 그대로 사용 가능하다.
- **코루틴 (Coroutines):** 비동기 프로그래밍을 마치 코드 짜듯이 쉽게 만들 수 있게 해준다.
- **멀티플랫폼 (Kotlin Multiplatform):** 이제는 안드로이드를 넘어 서버(Spring Boot), iOS, 웹 프런트엔드까지 하나의 코드로 로직을 공유할 수 있는 단계에 도달
