```kotlin
fun main() {  
    Outer.NestedClass("중첩중첩")  
    Outer("외부외부").InnerClass("내부내부")  
}  
  
class Outer(var name:String) {  
    class NestedClass(var name:String) {  
        init {  
            println("중첩 클래스: $name")  
        }  
    }  
  
    inner class InnerClass(var name:String) {  
        init {  
            println("내부 클래스 $name")  
            println("내부 클래스에서 외부 클래스 호출 ${this@Outer.name}")  
        }  
    }  
}
```

자바에서는 디폴트가 내부 클래스이고 `static`을 사용하면 중첩 클래스이다.
