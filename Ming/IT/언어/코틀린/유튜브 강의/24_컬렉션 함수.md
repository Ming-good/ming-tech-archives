#### `partition()`
> 아이템에 조건울 걸어 두개의 컬렉션으로 나누어 줌
![[Pasted image 20260218184639.png]]

```kotlin
fun groupFun() {  
    val listOf = listOf(  
        Person("유나", 1982),  
        Person("조이", 1996),  
        Person("츄", 1999),  
        Person("유나", 2003)  
    )  
    val associateBy = listOf.associateBy { it.age }  
    println(associateBy[1982]?.name)  
  
    val groupby = listOf.groupBy { it.name }  
    println(groupby["유나"]?.get(0)?.introduce())  
  
    val (a, b)= listOf.partition { it.age >= 1999 }  
    println(a)  
    println(b)  
}
```

fun transFun() {  
    val list = listOf(1, 2, 3, 4, 5, 6)  
    println(list.flatMap { listOf(it * 10, it + 10) })  
  
    println(list.getOrElse(1){100})
