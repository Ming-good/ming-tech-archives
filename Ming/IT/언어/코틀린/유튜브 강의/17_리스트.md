```kotlin
fun main() {  
    val listof = listOf<Int>(1, 2, 3)  
    println(listof[0])  
  
    val mutableListOf = mutableListOf<String>("A", "B", "C")  
    println(mutableListOf)  
  
    mutableListOf.add("D")  
    println(mutableListOf)  
  
    mutableListOf.add(1,"E")  
    println(mutableListOf)  
  
    mutableListOf.removeAt(1)  
    println(mutableListOf)  
}
```

- `listOf()` = 수정 불가
- `mutableListOf()` = 수정 가능
