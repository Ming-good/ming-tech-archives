## 상수 선언

> `val`와 `const val` 차이

| **구분**    | **val**                       | **const val**                        |
| --------- | ----------------------------- | ------------------------------------ |
| **결정 시점** | 런타임 (Runtime)                 | 컴파일 타임 (Compile-time)                |
| **자바 대응** | `private final` + `getter` 생성 | `public static final` (Field만 존재)    |
| **허용 타입** | 모든 객체 및 타입                    | 기본 자료형(Primitive), String            |
| **선언 위치** | 지역 변수, 클래스 멤버 등 어디든 가능        | 최상위(Top-level), `companion object` 내 |
- **인라이닝(Inlining):** `const val`은 컴파일 시점에 사용처에 값이 직접 치환되어 메서드 호출 오버헤드가 **없음**.
- **Getter 생성:** `val`은 자바 변환 시 내부적으로 Getter 메서드가 생성되어 호출 시마다 메서드를 거치게 **됨**.
- **제약 사항:** 애노테이션의 인자처럼 컴파일 타임에 값이 확정되어야 하는 곳에는 `const val`만 사용 가능함.
## 지연 선언
> `lateinit var`
- **정의:** 선언과 동시에 초기화하지 않고, 이후에 값을 할당하겠다고 명시하는 방식임.
- **특징:**
    - `var` 키워드와 함께 사용하며, 초기화 전 접근 시 예외가 **발생함**.
    - Primitive 타입(Int, Long 등) 및 Nullable(`?`) 타입에는 사용할 수 없는 제약이 **있음**.
    - 주로 스프링의 의존성 주입(`@Autowired`)이나 테스트 프레임워크의 객체 생성 시점에 활용함.
    - `::property.isInitialized`를 통해 초기화 여부를 런타임에 확인할 수 **있음**.
## 지연 대리자 속성
> `by lazy`
- **정의:** 해당 변수에 실제로 처음 접근하는 시점에 초기화 블록이 실행되는 방식임.
- **특징:**
    - `val` 키워드와 함께 사용하며, 초기화된 값을 내부적으로 캐싱하여 재사용함.
    - 기본적으로 **Thread-safe**하게 설계되어 멀티스레드 환경에서 안전성이 **있음**.
    - 초기화 로직이 무겁거나, 특정 상황에서만 사용되는 객체의 성능 최적화에 유리함.
```kotlin
  
fun main() {  
    FootCourt.search(FootCourt.FOOD_AGG)  
  
    println(LateInitSample.getLateInitText())  
    LateInitSample.text = "헬로우"  
    println(LateInitSample.getLateInitText())  
  
    val a: Int by lazy {  
        println("초기화 됩니다")  
        40+40  
    }  
  
  
    println("아직 초기화 안 됨")  
    println(a)  
}  
  
class FootCourt {  
    companion object{  
        const val FOOD_AGG = "TESTTREST"  
        const val FOOD_THRASH = "WEEEEEG!!"  
        fun search(foodName:String) {  
            val i = when (foodName) {  
                FOOD_AGG -> 13000  
                FOOD_THRASH -> 13000  
                else -> 0  
            }  
            println("$foodName 이다 이 음식은 가격이 $i 인데!!!!")  
        }  
    }  
  
}  
  
object LateInitSample {  
    lateinit var text:String  
    fun getLateInitText():String {  
        if (::text.isInitialized) {  
            return text  
        } else{  
            return "기본값"  
        }  
    }  
}
```
