1. **`just()`**
    - 가변 인자 입력 방식.
    - **null 요소 포함 불가** → null 포함 시 `NullPointerException` 발생.
2. **`justOrEmpty()`**
	- null요소 포함 허용
	- null이 전달되면 내부적으로 `empty()`를 호출
	- NullPointerException없이 onComplete signal만 전송한다.
3. **`empty()`**
    - emit할 데이터 없이 곧바로 `onComplete` 신호 전송.
    - 데이터는 없지만 작업 종료를 알리고 후처리를 할 때 사용.
4. **`fromArray()`**
    - 배열을 Flux로 변환.
    - **null 요소 포함 가능** → null 요소는 emit하지 않음.
5. **`fromIterable()`**
	- `List`, `Set`, `Queue`, 기타 Iterable 구현체들을 Flux로 변환
	- **null 요소 포함 가능** → null 요소는 emit하지 않음.
	- 리스트의 데이터를 순차적으로 emit
6. `fromStream()`
	- Stream에 포함된 데이터를 emit하는 Flux를 생성한다.
7. `range(n, m)`
	- n부터 1씩 증가하는 연속된 수를 m개 emit하는 Flux를 생성한다.
	- `IntStream.range()`와 달리 m은 (n+m-1) 개수만큼 처리 된다.
8. `defer()`
	- 구독하는 시점에 데이터를 emit하는 지연평가 함수이다.
9. `using(resource, emit flux, resource close)`

```java
System.out.println("=== FLUX EX1 ===");  
Flux.just(6, 9, 13)  
	.map(num -> num % 2)  
	.subscribe(System.out::println);  
	  
System.out.println("=== FLUX EX2 ===");  
Flux.fromArray(new Integer[]{3, 6, 7, 9})  
	.filter(num -> num > 6)  
	.map(num -> num * 2)  
	.subscribe(System.out::println);  
```
