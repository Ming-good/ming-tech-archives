```java
StepVerifier  
	.withVirtualTime(() -> TimeBasedTestEx.getCOVID19Count(  
			Flux.interval(Duration.ofHours(1)).take(2)  
		)  
	)  
	.expectSubscription()  
	.then(() -> VirtualTimeScheduler  
					.get()  
					.advanceTimeBy(Duration.ofHours(2))
	)  
	.expectNextCount(22)  
	.expectComplete()  
	.verify();
```
- `withVirtualTIme()`: `VirtualTimeScheduler`라는 가상의 스케줄러의 제어를 받도록 해 줌.
- `VirtualTimeScheduler.advanceTimeBy()`: 시간을 앞당기는 작업을 수행

```java
StepVerifier  
	.create(TimeBasedTestEx.getCOVID19Count(  
		Flux.interval(Duration.ofMinutes(1))
			.take(1)  
	))  
	.expectSubscription()  
	.expectNextCount(11)  
	.expectComplete()  
	.verify(Duration.ofSeconds(3));
```
- `verify(Duration.OfSeconds(3))`: 기대값의 평가가 시간 이내로 이루어지지 않는다면 `AssertionError`가 발생한다.

```java
StepVerifier  
	.withVirtualTime(() -> TimeBasedTestEx.getVoteCount(  
		Flux.interval(Duration.ofMinutes(2))  
	))  
	.expectSubscription()  
	.thenAwait(Duration.ofMinutes(8))  
	.expectNoEvent(Duration.ofMinutes(1))  
	.expectNoEvent(Duration.ofMinutes(1))  
	.expectNextCount(5)  
	.expectComplete()  
	.verify();
```
- `expectNoEvent()`: 시간을 지정하면 지정한 시간동안 어떤 이벤트도 발생하지미 않을 것을 기대하며 지정한 시간만큼 시간을 앞당긴다
- `thenAwait()` : 시간을 지정하면 시간만 앞당긴다
- 해설: 
	- `TimeBasedTestEx.getVoteCount` : 총 5개의 데이터를 만든다
	- `Flux.interval(Duration.ofMinutes(2))`:  10분 뒤에 작업이 완료된다.
	- `thenAwait`로 8분이 지났으며 `expectNoEvent`로 2분 동안 어떤 이벤트도 발생하지 않으며 시간이 지날 것을 기대한다.
	- `expectNextCount` : 총 5개의 데이터가 emit되었음을 기대한다
	- `expectComplete` : 작업이 종료되었을 것을 기대한다.
