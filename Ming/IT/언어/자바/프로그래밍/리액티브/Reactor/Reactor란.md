- 스프링 프레임워크 팀의 주도하에 개발된 리액티브 스트림즈 구현체이다.
- 스프링 프레임워크5 버전부터 리액티브 스택에 포함되어 스프링 웹플럭스 기반의 리액티브 애플리케이션을 제작하기 위한 핵심 역할을 담당
- 리액티브 프로그래밍을 위한 라이브러리

## 특징
- **Reactive Streams**: Reactor는 리액티브 스트림즈 사양을 구현한 리액티브 라이브러리이다.
- **Non-Blocking** 
- 자바 함수형 API
- **`Flux[N]`**: Reactor의 Publisher타입중 하나로 0 ~ N개의 데이터를 emit한다는 의미이다. 
- **`Mono[0|1]`**: Reactor의 Publisher타입중 하나로 0 ~ 1개의 데이터를 emit한다는 의미이다.
- Reactor은 마이크로 서비스 기반 시스템에 적합한 기술이다.
- Backpressure를 지원
