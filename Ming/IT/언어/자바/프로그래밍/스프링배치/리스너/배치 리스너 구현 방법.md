## 인터페이스 구현
```java
@Slf4j  
@Component  
public class JobExecutionListenerConfig implements JobExecutionListener {  
  
	@Override  
	public void beforeJob(JobExecution jobExecution) {  
		log.info("시스템 감시 시작. 모든 작업을 내 통제 하에 둔다.");  
	}  
	@Override  
	public void afterJob(JobExecution jobExecution) {  
		log.info("작업 종료. 할당된 자원 정리 완료");  
		log.info("시스템 상태: {}", jobExecution.getStatus());  
	}  
}

@Slf4j  
@Component  
public class StepExecutionListenerConfig implements StepExecutionListener {  
  
	@Override  
	public void beforeStep(StepExecution stepExecution) {  
		log.info("Steprndur 감시 시작. 모든 행동이 기록된다.");  
	}  
	  
	@Override  
	public ExitStatus afterStep(StepExecution stepExecution) {  
		log.info("Step 감시 종료. 모든 행동이 기록되었다");  
		return ExitStatus.COMPLETED;  
	}  
}
```

## 어노테이션 기반 구현
```java
@Slf4j  
@Component  
public class AnnotationListenerConfig {  
	@BeforeJob  
	public void start(JobExecution jobExecution) {  
		log.info("====== Job 리스너 실행 ======");  
	}  
	  
	@AfterJob  
	public void end(JobExecution jobExecution) {  
		log.info("====== Job 리스너 종료 ======");  
	}  
	  
	@BeforeStep  
	public void stepStart(StepExecution stepExecution) {  
		log.info("===> Step 리스너 시작");  
	}  
	  
	@AfterStep  
	public ExitStatus stepEnd(StepExecution stepExecution) {  
		log.info("===> Step 리스너 종료");  
		return ExitStatus.COMPLETED;  
	}  
}
```

## 호출 방법
```java
/* 인터페이스 기반 */
@Bean  
public Job systemMinotoringJob(JobRepository repository, Step monitoringStep) {  
	return new JobBuilder("systemMonitoringJob", repository)  
				.listener(new JobExecutionListenerConfig())  
				.start(monitoringStep)  
				.build();  
}

@Bean  
public Step monitoringStep(JobRepository repository) {  
	return new StepBuilder("monitoringStep", repository)  
				.listener(new StepExecutionListenerConfig())  
				.tasklet((contribution, chunkContext) -> {  
					log.info("스텝 진행중");  
					return RepeatStatus.FINISHED;  
				}, transactionManager)  
				.build();  
}


/* 어노테이션 기반 */
@Bean  
public Job annotationMinotoringJob(JobRepository repository, Step annotationMonitoringStep) {  
	return new JobBuilder("annotationMinotoringJob", repository)  
				.listener(new AnnotationListenerConfig())  
				.start(annotationMonitoringStep)  
				.build();  
}  
  
@Bean  
public Step annotationMonitoringStep(JobRepository repository) {  
	return new StepBuilder("annotationMonitoringStep", repository)  
				.listener(new AnnotationListenerConfig())  
				.tasklet((contribution, chunkContext) -> {  
					log.info("어노테이션 스텝 진행중");  
					return RepeatStatus.FINISHED;  
				}, transactionManager)  
				.build();  
}
```
