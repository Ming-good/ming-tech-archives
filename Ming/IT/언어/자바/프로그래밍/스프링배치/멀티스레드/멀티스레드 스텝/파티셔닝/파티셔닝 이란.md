## 파티셔닝이란?
- **서로 다른 데이터 범위(파티션)를 처리하는 여러 개의 독립적인 스텝 실행을 동시에 수행하는 것'**
- 각 파티션을 처리하는 **각각의 스레드는 완전히 별도의 StepExecution으로실행**된다

## 실행 순서
1. `StepExecutionSplitter`가 데이터를 범위별로 분할한다.
	- 각 파티션마다 독립적인 워커 `StepExecution`인스턴스를 생성한다.
2. `PartitionHandler`가 실질적인 로직을 실행한다.
	1. `StepExecutionSplitter`가 생성한 워커 `StepExecution`을 입력 받아서
	2. 각각 스레드에 할당하여 병렬 실행을 명령한다.
	3. 모든 워커 스텝의 작업이 완료되면, 각 워커 실행결과를 취합, 정리한다.
> 각 파티션마다 `ExecutionContext`를 가지며 이를 바탕으로 파티션별 `StepExecution`을 생성한다.
