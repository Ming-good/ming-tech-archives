## 플랫 파일이란?
- 행과 열로만 구성된 파일을 플랫파일이라고 한다.
- 각 라인이 하나의 데이터다
---

## `FlatFileItemReader`
#### 핵심 동작
- 파일에서 한 줄 읽기
- 읽어온 문자열을 객체로 변환
- `LineMapper`를 이용해서 파일 한 줄을 객체로 변환하는 역할을 한다.
#### `LineMapper`
```java
public interface LineMapper<T> { 
	T mapLine(String line, int lineNumber) throws Exception; 
}
```
- 문자열을 객체로 변환하는 역할을 담당
- [[DefaultLineMapper]] : `LineMapper`의 구현체
