## `CompositeItemWriter` -  순차실행
> 하나의 ItemProcessor 처리 결과를 여러 데이터 소스에 동시에 저장해야 할 때 사용
#### 구조
```java
public void write(Chunk<? extends T> chunk) throws Exception {
    for (ItemWriter<? super T> writer : delegates) {
       writer.write(chunk);
    }
}
```
`ItemWriter`들을 위임 대상으로 등록해놓으면, `CompositeItemWriter`의 `writer()`메서드가 호출될 때마다 모든 위임 대상 `ItemWriter`들이 차례대로 실행된다.
>  즉, 파일쓰기도 하면서 메시지를 보낸다던지 다른 쓰기 방식을 함께 처리할 수 있다.

![[Pasted image 20251019002501.png]]
#### 사용법
```java
// 1. 생성자에 직접 Writer들을 주입
CompositeItemWriter<Hacker> writer = new CompositeItemWriter<>(
   List.of(
       firstWriter,
       secondWriter,
       thirdWriter
   )
);

// 2. Builder의 delegates() 메서드 활용
CompositeItemWriter<Hacker> writer = new CompositeItemWriterBuilder<Hacker>()
   .delegates(List.of(
       firstWriter,
       secondWriter,
       thirdWriter
))
.build();
```

## `ClassifierCompositeItemWriter` - 분류 처리
1. 청크 단위로 들어온 데이터를 하나씩 분류(`classify`)한다
2. 분류된 결과에 따라 적절한 위임 대상 ItemWriter에 item을 매핑한다
3. 청크의 모든 item이 분류되면, 각 ItemWriter가 자기에게 할당된 item을 처리한다.
#### 구조
![[Pasted image 20251019003933.png]]
```java
private Classifier<T, ItemWriter<? super T>> classifier = new ClassifierSupport<>(null);
```

#### 사용 예시
- ClassifierCompositeItemWriter 객체를 생성하고 `setClassifier()` 메서드에 분류기를 장착
- `ClassifierCompositeItemWriterBuilder.classifier()`로 한 번에 조립

```java
private Step systemLogProcStep() {  
	return new StepBuilder("systemLogProcStep", repository)  
		.<SystemLog, SystemLog>chunk(10, transactionManager)  
		.reader(systemLogProceReader())  
		.writer(classifierWriter())  
		.build();  
}

@Bean  
public ClassifierCompositeItemWriter<SystemLog> classifierWriter() {  
	ClassifierCompositeItemWriter<SystemLog> writer = new ClassifierCompositeItemWriter<>();  
	writer.setClassifier(
		new SystemLogClassifer(criticalLogWritor(), normalLogWriter())
	);  
	return writer;  
}  
  
private ItemWriter<SystemLog> normalLogWriter() {  
	return items -> {  
		log.info("특별 로그 처리");  
		for (SystemLog logs : items) {  
			log.info("특별 처리: {}", logs);  
		}  
	};  
}  
  
private ItemWriter<SystemLog> criticalLogWritor() {  
	return items -> {  
		log.info("로그 처리중");  
		for (SystemLog logs : items) {  
			log.info("일반 처리: {}", logs);  
		}  
	};  
}

public static class SystemLogClassifer implements Classifier<SystemLog, ItemWriter<? super SystemLog>> {  
	public static final int CRITICAL_CPU_THRESHOLD = 90;  
	public static final long CRITICAL_MEMORY_THRESHOLD = 1024 * 1024 * 1024;  
	private final ItemWriter<SystemLog> systemLogItemWriter;  
	private final ItemWriter<SystemLog> criticalLogWritor;  
  
	public SystemLogClassifer(ItemWriter<SystemLog> systemLogItemWriter, ItemWriter<SystemLog> criticalLogWritor) {  
		this.systemLogItemWriter = systemLogItemWriter;  
		this.criticalLogWritor = criticalLogWritor;  
	}  
  
	@Override  
	public ItemWriter<? super SystemLog> classify(SystemLog classifiable) {  
		if ("CRITICAL".equals(classifiable.getType())) {  
			return criticalLogWritor;  
		}  
			return systemLogItemWriter;  
		}  
	}  
}
```
