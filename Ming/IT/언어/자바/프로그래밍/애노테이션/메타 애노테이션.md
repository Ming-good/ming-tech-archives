> 애노테이션을 정의하는데 사용하는 특별한 애노테이션을 메타 애노테이션이라 한다.

```java
@Documented
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.ANNOTATION_TYPE)
public @interface Retention {
	RetentionPolicy value();
}
```
## @Rettention
- 애노테이션의 생존 기간을 지정한다.
```java
public enum RetentionPolicy {
	SOURCE,
	CLASS,
	RUNTIME
}

```
- `RetentionPolicy.SOURCE` : 소스 코드에만 남아있다. 컴파일 시점에 제거된다.
- `RetentionPolicy.CLASS` : 컴파일 후 class 파일까지는 남아있지만 자바 실행 시점에 제거된다. (기본 값)
- `RetentionPolicy.RUNTIME` : 자바 실행 중에도 남아있다. 대부분 이 설정을 사용한다.
## @Target
- 애노테이션 적용할 수 있는 위치를 지정한다.
```java
public enum ElementType {
	TYPE,
	FIELD,
	METHOD,
	PARAMETER,
	CONSTRUCTOR,
	LOCAL_VARIABLE,
	ANNOTATION_TYPE,
	PACKAGE,
	TYPE_PARAMETER,
	TYPE_USE,
	MODULE,
	RECORD_COMPONENT;
}
```

## @Documented
- 자바 API문서를 만들 때 해당 애노테이션이 함께 포함되는지 지정한다. 보통 함께 사용.
## @Inherited
- 자식 클래스가 애노테이션을 상속 받을 수 있다.
