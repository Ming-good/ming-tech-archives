>수정자는 접근 제어자와 비 접근 제어자(기타 수정자)로 나눌 수 있다.
- 접근 제어자: `public` , `protected` , `default` (`package-private` ), `private`
- 비 접근 제어자: `static` , `final` , `abstract` , `synchronized` , `volatile` 등

#### 메서드 메타데이터
- `getMethods()` : 해당 클래스와 상위 클래스에서 상속된 모든 **public메서드를 반환**
- `getDeclaredMethods()` : **해당 클래스에 선언된 모든 메서드를 반환**하며, 접근 제어자에 관계없이 반환. 상속된 메서드는 포함하지 않음

#### 동적 메서드 호출
```java
// 정적 메서드 호출 - 일반적인 베서드 호출  
BasicData basicData = new BasicData();  
basicData.call(); // 이부분은 코드를 변경하지 않는 이상 정적이다.  
  
// 동적 메서드 호출 - 리플렉션 사용  
Class<? extends BasicData> aClass = basicData.getClass();  
String methodName = "hello";  
  
/**  
* 메서드 이름을 변수로 변경할 수 있다.  
* (public 메서드는 모두 호출 가능)  
* (상속된 메서드는 포함)  
*/  
Method method = aClass.getMethod(methodName, String.class);  
Object result0 = method.invoke(basicData, "public은 전부 호출 가능함. 상속된 메서드도 포함");  
System.out.println("result0 = " + result0);  
  
/**  
* 메서드 이름을 변수로 변경할 수 있다.  
* (해당 클래스에 선언된 모든 메서드 중 조회 후 반환)  
* (상속된 메서드는 포함 안 됨.)  
*/  
Method hello = aClass.getDeclaredMethod(methodName, String.class);  
Object result = hello.invoke(basicData, "안녕하세요. 첫번째 인자 입니다.");  
System.out.println("result = " + result);
```
