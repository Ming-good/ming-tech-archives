- 컴퓨터 과학에서 사용하는 **원자적 연산**(atomic operation)의 의미는 해당 연산이 **더 이상 나눌 수 없는 단위로 수행**된 다는 것을 의미한다. 
>즉, 원자적 연산은 **중단되지 않고**, 다른 연산과 간섭 없이 **완전히 실행되거나 전혀 실행되지 않는 성질**을 가지고 있다.  쉽게 이야기해서 **멀티스레드 상황**에서 다른 스레드의 간섭 없이 **안전하게 처리되는 연산**이라는 뜻이다.

##### 원자적 연산 예시
```java
volatile int i = 0;

i = 1    // 원자적 연산
```
1. 오른쪽에 있는 `1` 의 값은 왼쪽의 `i` 변수에 대입한다.
##### 비원자적 연산 예시
```java
volatile int i = 0;

i = i + 1; // 원자적 연산이 아니다
```

1. 오른쪽에 있는 `i` 의 값을 읽는다. `i` 의 값은 10이다.
2. 읽은 10에 1을 더해서 11을 만든다.
3. 더한 11을 왼쪽의 `i` 변수에 대입한다.
