
## 단방향 관계
#### `@ManyToOne`
![[Pasted image 20260311142726.png]]
```kotlin
@Entity  
@Table(name = "STUDENT_TB")  
class Student(  
    @Id  
    @GeneratedValue    
    var studentId:Long?=null,  
    var name:String?=null,  
    var grade:String?=null,  
  
    @ManyToOne  
    @JoinColumn(name = "MAJORID")  
    var major:Major?=null  
)

---

val major = Major(name = "Computer Science", category = "College of Engineering")  
em.persist(major)  

val student = Student(name = "Kim", grade = "3")  
student.major = major  
em.persist(student)

```
#### FetchType
- `@ManyToOne(fetch = FetchType.LAZY)` : 연관 객체를 호출하는 순간 SELECT 조회를 실행함
- `@ManyToOne(fetch = FetchType.EAGER)` :  연관 객체를 즉시 SELECT 조회 실행함


## 양방향 관계
## `@OneToMany`
```kotlin
@Entity  
class CountryEntity (  
    @Id  
    @GeneratedValue(strategy= GenerationType.IDENTITY)  
    var id: Long? = null,  
  
    @Column(unique = true)  
    var country:String? = null,  
  
    @OneToMany(mappedBy = "countryEntity")  
    var cityEntities: MutableList<CityEntity>? = null  
) {  
    fun addCityEntity(cityentity:CityEntity) {  
        cityEntities?.add(cityentity)  
        cityentity.countryEntity=this  
    }  
}
```
- `mappedBy = "상대 칼럼명"`:   양방향 관계에서 이 필드는 내가 직접 관리하는 게 아니라, 저쪽 필드에 매핑되어 있는 거울일 뿐이야라고 선언하는 속성
